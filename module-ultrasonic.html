單元教學：HC-SR04 超音波感測器

本單元介紹如何利用超音波反射的時間差，計算出前方障礙物的具體距離，常用於避障車與自動鐵捲門專案。

1. 準備材料

  • ESP32 開發板 x1
  • HC-SR04 超音波模組 x1
  • 麵包板與母對公杜邦線 若干

2. 接腳與接線對照

HC-SR04 腳位 ESP32 開發板端 說明
VCC VIN / 5V 驅動超音波需要 5V 電壓
Trig (觸發) GPIO 5 由 ESP32 發射觸發訊號
Echo (接收) GPIO 18 接收反射訊號（*建議串電阻作 5V 到 3.3V 分壓）
GND GND 共地

4. Arduino IDE 範例程式碼

const int trigPin = 5;
const int echoPin = 18;

// 定義聲音在 20°C 時的速度為 0.0343 公分/微秒
#define SOUND_SPEED 0.0343

long duration;
float distanceCm;

void setup() {
  Serial.begin(115200); // 啟動序列埠監控器
  pinMode(trigPin, OUTPUT); 
  pinMode(echoPin, INPUT);  
}

void loop() {
  // 清空 Trig 腳位電位
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  
  // 觸發發射超音波（持續 10 微秒的高電位）
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  // 讀取 Echo 腳位返回脈衝的時間（微秒）
  duration = pulseIn(echoPin, HIGH);
  
  // 計算距離 (時間 * 音速 / 2)
  distanceCm = duration * SOUND_SPEED / 2;
  
  // 將結果輸出至序列埠監控器
  Serial.print("距離 (cm): ");
  Serial.println(distanceCm);
  
  delay(500); // 每 0.5 秒量測一次
}
