單元教學：外接發光二極體 (LED)

本單元將帶領您學習如何透過 ESP32 的數位輸出（Digital Output）功能控制外部 LED 的亮滅。

1. 準備材料

  • ESP32 開發板 x1
  • 5mm 紅色 LED x1
  • 220歐姆 電阻 x1
  • 麵包板與杜邦線 若干

2. 接腳與接線對照

LED 元件端 ESP32 開發板端 說明
長腳 (正極 Anode) GPIO 23 透過串聯 220 歐姆電阻連接至數位輸出腳
短腳 (負極 Cathode) GND 連接至系統接地端

3. 接線範例圖

【請在此放入 Fritzing 繪製的 ESP32 GPIO23 -> 電阻 -> LED -> GND 接線圖】

4. Arduino IDE 範例程式碼

請複製以下程式碼至 Arduino IDE 中，並選擇正確的 ESP32 開發板進行燒錄：

// 定義 LED 接腳為 GPIO 23
const int ledPin = 23;

void setup() {
  // 設定該腳位為輸出模式
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH); // 輸出高電位，點亮 LED
  delay(1000);                // 等待 1 秒 (1000毫秒)
  
  digitalWrite(ledPin, LOW);  // 輸出低電位，熄滅 LED
  delay(1000);                // 等待 1 秒
}
